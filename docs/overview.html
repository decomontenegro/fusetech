FuseLabs App

Plataforma de tokenização para atividades físicas e sociais

Visão Geral da Arquitetura

O FuseLabs App é uma aplicação distribuída que segue uma arquitetura de microsserviços. A aplicação é dividida em vários serviços independentes que se comunicam entre si através de APIs REST, GraphQL e mensagens assíncronas via Redis.

Componentes Principais

Frontend (Web)

Interface de usuário construída com Next.js, React e TailwindCSS. Responsável pela autenticação de usuários, exibição de dados e integração com carteiras de criptomoedas.

Next.js React TailwindCSS Apollo Client

Token Service

Serviço responsável pela interação com a blockchain Base L2, mint e burn de tokens ERC-20, gerenciamento de transações e exposição de API para consulta de saldos.

Node.js Fastify ethers.js Redis

Strava Worker

Serviço responsável pela integração com a API do Strava, processamento de webhooks, verificação e validação de atividades físicas e cálculo de pontos.

Node.js Fastify Redis

Social Listener

Serviço responsável pela integração com APIs de redes sociais, monitoramento de hashtags e menções, verificação de posts sociais e cálculo de pontos.

Node.js Fastify Redis

Fraud Detection

Serviço responsável pela detecção de atividades fraudulentas, análise de padrões de atividades físicas e verificação de autenticidade de posts sociais.

Node.js Fastify ML Libraries

Gamification

Serviço responsável pelo gerenciamento de desafios, conquistas, leaderboards e sistema de níveis e recompensas.

Node.js Fastify Redis
